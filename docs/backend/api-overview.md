# Backend API

- `POST /api/transcribe` — SSE ile transkript oluşturma
- `GET /api/transcripts/{id}` — kayıtlı transkript
- `POST /api/translate` — SSE ile çeviri
- `POST /api/generate-prompt` — AI prompt üretimi
- `POST /api/summarize` — video özeti
- `GET /api/history` — geçmiş
- `GET /api/models` — Ollama modelleri
- `GET /api/export/{id}/{format}` — txt/srt/vtt
